## Koneksi eps8266 dan sensor ultrasonik
Dari gambar di atas dapat dilihat pengkabelan seperti pada tabel di bawah ini

| ESP8266 Amica | Sensor Ultrasonic                  |
|---------------|------------------------------------|
| VCC           | Vcc                                |
| GND           | Gnd                                |
| D1            | Trig                               |
| D2            | Echo                               |
